# Career-Compass
A personalized AI-driven platform that recommends suitable career paths using Google’s Gemini API. It analyzes users’ skills, interests, and backgrounds to provide tailored guidance, along with relevant certifications and learning resources from trusted sites like Udemy and Coursera for informed career decisions.

# Running the code
Run python server.py to run backend python server.
Run npm install to install the dependencies.
Run npm run dev to start the development server.
